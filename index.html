Engagement Process

계약 → Deploy까지 4단계

평균 4-12주. 30% 선급 / 70% 완료 결제 표준. 진행 중 weekly 진척도 보고.

01

요구사항 미팅

1-2시간 무료 상담. 회사 도메인·예산·일정 파악 → 솔루션 매칭.

02

견적·계약

견적서 1-3일 내 발송. 계약서 + NDA + 30% 선급 → 착수.

03

Deploy

2-8주. 우리 코드 fork → 회사 브랜딩·도메인 적용 → staging → production.

04

인계·운영

코드 인계 + 문서 + 운영 가이드. 월 단위 유지보수 retainer 옵션.
