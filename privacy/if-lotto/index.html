개인정보처리방침

시행일: 2026년 3월 26일 | 만약에로또 (com.reaf.if_lotto)

Hundred Core(이하 "회사")는 「개인정보 보호법」에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 다음과 같이 개인정보처리방침을 수립하여 공개합니다.

1. 수집하는 개인정보 항목

1.1 자동 수집 항목

항목수집 목적보관 기간
익명 사용자 식별자 (Firebase Anonymous UID)사용자 구분 및 데이터 연동앱 삭제 시 삭제
FCM 푸시 토큰당첨 결과 알림 발송앱 삭제 시 삭제

1.2 이용자 제공 항목

  • 로또 번호 생성 기록: 추천 번호, 생성 방식(랜덤/균형/핫넘버/콜드넘버/꿈해몽) - Firestore에 저장
  • 꿈해몽 입력 텍스트: AI 번호 추천에 사용 후 즉시 폐기

1.3 수집하지 않는 정보

회사는 이름, 이메일, 전화번호 등 개인을 직접 식별할 수 있는 정보를 수집하지 않습니다.

  • 실명, 이메일 주소, 전화번호
  • 위치 정보 (GPS)
  • 연락처, 통화 기록
  • 결제 정보 (Google Play에서 직접 처리)

2. 카메라 권한

본 앱은 QR 코드 스캔 기능을 위해 카메라 접근 권한을 요청할 수 있습니다.

  • 카메라로 촬영된 이미지는 기기 내에서만 처리되며 서버에 업로드되지 않습니다
  • QR 코드 인식 후 이미지 데이터는 즉시 폐기됩니다
  • 카메라 권한은 선택사항이며, 거부해도 앱의 핵심 기능을 사용할 수 있습니다
  • 카메라 권한은 기기 설정에서 언제든지 변경할 수 있습니다

3. 개인정보의 수집 및 이용 목적

  • 로또 번호 추천 및 당첨 대조 서비스 제공
  • 누적 당첨 통계 집계 (익명 데이터 기반)
  • 당첨 결과 푸시 알림 발송 (FCM)
  • 서비스 개선 및 이용 통계 분석
  • 광고 게재 (Google AdMob)

4. 개인정보의 보유 및 이용 기간

  • 당첨되지 않은 예측 기록: 대조 완료 후 30일 이내 자동 삭제
  • 당첨된 예측 기록: 누적 통계 목적으로 익명화하여 보관
  • FCM 푸시 토큰: 앱 삭제 또는 토큰 갱신 시 자동 교체
  • 앱 삭제 시: 기기 내 저장 데이터 즉시 삭제, 서버 데이터 삭제 요청 가능

5. 개인정보의 제3자 제공

회사는 이용자의 개인정보를 원칙적으로 제3자에게 제공하지 않습니다. 단, 다음의 경우는 예외로 합니다:

  • 법령에 의해 요구되는 경우
  • 광고 서비스 제공을 위한 Google AdMob (광고 식별자만 전달)

6. 제3자 서비스

본 앱은 다음의 제3자 서비스를 이용합니다:

  • Firebase Authentication (Google LLC) - 익명 인증. Firebase 개인정보처리방침
  • Firebase Firestore (Google LLC) - 번호 예측 기록 저장
  • Firebase Cloud Messaging (FCM) (Google LLC) - 당첨 결과 푸시 알림
  • Firebase Analytics & Crashlytics (Google LLC) - 앱 분석 및 오류 보고
  • Google AdMob (Google LLC) - 광고 게재. Google 개인정보처리방침

7. 광고

본 앱은 Google AdMob을 통해 배너 광고, 전면 광고, 리워드 광고를 게재합니다. AdMob은 광고 제공을 위해 광고 식별자 및 기기 정보를 수집할 수 있습니다.

  • 광고 개인화는 기기 설정(설정 > 개인정보 > 광고)에서 비활성화할 수 있습니다
  • 리워드 광고 시청은 선택사항이며, 추가 번호 생성 혜택을 제공합니다

8. 이용자의 권리

  • 앱 삭제를 통해 기기 내 모든 로컬 데이터를 삭제할 수 있습니다
  • 서버에 저장된 데이터 삭제를 이메일로 요청할 수 있습니다
  • 광고 개인화를 기기 설정에서 비활성화할 수 있습니다
  • 푸시 알림을 기기 설정에서 비활성화할 수 있습니다

9. 개인정보의 안전성 확보 조치

  • 데이터 전송 시 SSL/TLS 암호화 적용
  • Firebase 보안 규칙을 통한 접근 제어
  • 익명 인증을 통한 최소 정보 수집 원칙 적용
  • 카메라 데이터의 기기 내 처리 (서버 미전송)

10. 아동의 개인정보 보호

만약에로또는 만 19세 이상 성인을 대상으로 합니다. 미성년자의 복권 구매는 법적으로 금지되어 있습니다. 회사는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.
