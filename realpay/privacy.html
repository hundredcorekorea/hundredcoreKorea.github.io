개인정보처리방침

RealPay 시급지킴이

최종 수정일: 2026년 2월 1일

요약: RealPay 시급지킴이는 이용자의 개인정보를 수집하지 않습니다. 모든 데이터는 이용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

1. 개인정보 처리 원칙

HundredCore Korea(이하 "회사")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보처리방침은 회사가 제공하는 RealPay 시급지킴이 앱(이하 "앱")의 개인정보 처리에 관한 사항을 규정합니다.

2. 수집하는 개인정보

중요: 본 앱은 어떠한 개인정보도 수집하지 않습니다.

2.1 앱에서 사용되는 데이터

다음 데이터는 이용자의 기기에만 저장되며, 회사 서버로 전송되지 않습니다:

  • 프로젝트 정보 (프로젝트명, 계약금액, 목표 시급 등)
  • 클라이언트 정보 (클라이언트명, 연락처 메모 등)
  • 작업 시간 기록
  • 앱 설정 (다크 모드, 알림 설정 등)

2.2 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 기기 고유 식별자
  • 사용 패턴 또는 행동 데이터

3. 데이터 저장 및 보안

항목 저장 위치 보안 방법
프로젝트 데이터 기기 내부 저장소 앱 샌드박스 보호
앱 설정 기기 내부 저장소 앱 샌드박스 보호
공유 이미지 임시 저장소 자동 삭제 (1시간 후)

4. 제3자 서비스

4.1 인앱 결제

프리미엄 기능 결제 시 Google Play 또는 Apple App Store의 결제 시스템을 이용합니다. 결제 정보는 해당 플랫폼에서 직접 처리하며, 회사는 결제 정보에 접근하지 않습니다.

4.2 공유 기능

앱의 공유 기능을 사용할 경우, 이용자가 선택한 앱(카카오톡, 인스타그램 등)으로 이미지가 전달됩니다. 이는 이용자의 직접적인 행위에 의해서만 발생합니다.

4.3 광고 (해당되는 경우)

무료 버전에서 광고가 표시될 수 있습니다. 광고 네트워크(Google AdMob)는 자체적인 개인정보처리방침에 따라 데이터를 수집할 수 있습니다. 자세한 내용은 해당 서비스의 개인정보처리방침을 참조하세요:

  • Google 개인정보처리방침

5. 이용자의 권리

이용자는 다음과 같은 권리를 가집니다:

  • 데이터 삭제: 앱 삭제 시 모든 데이터가 기기에서 삭제됩니다.
  • 데이터 접근: 앱 내에서 모든 저장된 데이터를 열람할 수 있습니다.
  • 데이터 수정: 앱 내에서 모든 데이터를 자유롭게 수정할 수 있습니다.
  • 알림 거부: 기기 설정에서 앱 알림을 비활성화할 수 있습니다.

6. 아동의 개인정보

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

7. 개인정보처리방침의 변경

본 개인정보처리방침이 변경되는 경우, 변경 사항을 앱 내 공지 또는 본 페이지를 통해 안내합니다. 변경된 방침은 게시 즉시 효력이 발생합니다.

8. 데이터 보관 및 파기

1. 앱에서 생성된 모든 데이터는 이용자의 기기에만 저장됩니다.

2. 앱 삭제 시 모든 데이터가 자동으로 파기됩니다.

3. 공유 기능 사용 시 생성되는 임시 이미지 파일은 1시간 후 자동 삭제됩니다.

9. 국제 데이터 전송

본 앱은 이용자의 데이터를 외부 서버로 전송하지 않으므로, 국제 데이터 전송이 발생하지 않습니다. 단, 제3자 서비스(광고, 인앱 결제)를 통해 해당 서비스 제공자의 정책에 따라 데이터가 전송될 수 있습니다.
